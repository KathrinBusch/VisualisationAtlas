Overview

Visualisation in marine molecular ecology: Dendrograms, networks, maps

Kathrin Busch, Biological Oceanographer

What does the graphic show?:

The graphic is a bipartite network between sample type and microbial taxa, showing distributions of microbial taxa in deep-sea sponges and across sample types. Total numbers of micobial taxa occurring in each sample type are given and reflected by size of black dots. Edge and node colours refer to the prevalence of microbial taxa in one (yellow-beige), two (orange-salmon), all (red) sample types. The category “others” (grey) denotes microbial taxa that do not fall in any category.

Personal perspective:

Networks are one prime visualisation type in ecology. They display relations between elements, using simple links, and have a particular strength in revealing clusters and interactions. Ecological networks can be used to describe and compare the structure of ecosystems and properties such as ecosystem stability, making them particularly helpful tools for gaining a system perspective.

How was this visualization created?:

To generate this graph a combination of python and R (for generation of input data), gephi (for network visualisation), and inkscape (for graphical fine-tuning) was used.
