The Jupyter Ecosystem

Willi Rath, Data Scientist

What does the graphic show?:

The Jupyter Ecosystem consists of a variety of tools all circling about Literate Programming with Jupyter Notebooks. Literate Programming aims at integrating multiple aspects of a computational workflow into one document. These aspects often are code, formatted text such as documentation, alphanumerical outputs, and graphical outputs. The Jupyter Ecosystem provides tools and platforms for working with Jupyter Notebooks on the full range of computing and data platforms (from small calculations on a personal device to big data handling in the cloud or on supercomputers).

Personal perspective:

Jupyter allows for bundling all aspects of, e.g., a data analysis workflow in one place. I can outline the research question, show the code for ingesting and pre-processing the data, show some graphical output and discuss the results all in the same place. (Jupyter Notebooks don’t come without downsides, however: Often there’s parts of the code developed even in analysis workflows which are often re-used and hence should better be in a library or package. When developing workflows in Jupyter Notebooks, it’s often not clear when and which parts to migrate to a library.)
