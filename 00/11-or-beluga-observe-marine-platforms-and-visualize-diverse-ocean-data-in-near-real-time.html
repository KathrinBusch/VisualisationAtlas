Overview

BELUGA: Observe marine platforms and visualize diverse ocean data in near real-time

Patrick Leibold, Software-Engineer

with Sylvia Reißmann and Nikolaj Diller

What does the graphic show?:

Near real-time visualization of georeferenced ocean data

Monitoring and control of (autonomous) marine sensor platforms

Annotation and link of ocean data with blogs, social media, etc

Personal perspective:

Web-based display allows multiple instances on various types of terminals

Common map view for all platforms enables situational awareness

Dashboards and charts allow a detailed analysis of platform / ocean data

Additional media contributions from participating scientists allow an immersive experience of the scientific campaign

For more information take a look at the official BELUGA webpage, and the project webpage

How was this visualization created?:

Frontend:

Languages/Technologies: HTML, CSS, Javascript

Libraries/Frameworks: Vue.js, Bootstrap, Highcharts, Leaflet + various plugins

Backend:

Languages/Technologies: Python, PHP, WebSockets, MySQL, MariaDB

Libraries/Frameworks: ROS

Plattforms (AUV / Transponder):

Languages/Technologies: Python, C++, ROS

Libraries/Frameworks: ROS, COLA2
