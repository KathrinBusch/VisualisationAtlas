Overview

A web framework for dynamic data presentations in Earth Sciences

Everardo González, Data Scientist

What does the graphic show?:

The figure shows a screen-grab of a single presentation slide. All plots in this slide are dynamic and generated in real time by the presenter from raw data resources stored locally or online.

Personal perspective:

The last century has seen an unprecedented explosion in computational development which opened the gates to unforeseen and exciting advances in the fields of data and information science.

This comes in stark contrast to the standard Powerpoint (and similar) slides which emulate one-to-one the presentation style of the overhead projector, and so, limit the presenter’s expression, style, and content to that which was available and common some 70 years ago.

Web frameworks allow for the integration of contemporary data and computing techniques into presentation content. This takes the possibilities for scientific story telling to a whole new dimension.

How was this visualization created?:

Reveal.js is a framework for creating web presentations. Further web visualisation tools like D3.js were coupled with custom Javascript code and web embeddings to create rich visualisations from raw data.
